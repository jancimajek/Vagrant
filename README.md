Vagrant
=======
This project stores configuration for my Vagrant projects

Setup
=======
* `git clone https://github.com/jancimajek/Vagrant.git ~/Vagrant`
* Add the following to your `~/.bash_profile`: 
  `export VAGRANT_HOME=~/Vagrant/vagrant.d`
* `cd ~/Vagrant/project`
* `vagrant up`
